SolarVita

SolarVita is a fitness app that integrates eco friendly features, workout tracking, and meal planning. Built with Flutter, it supports multiple languages and cross platform optimization.

Savannah Bites

I developed a restaurant website with a MongoDB Atlas database and Node.js backend, focusing on seamless frontend-backend integration. There is also a implementation of a secure Stripe payment.

An AI powered platform that automates YouTube Shorts creation from scriptwriting to publishing. Features voice synthesis, video assembly, and analytics, all managed via a Flask based interface.
